Dojo Example

This example uses the Dojo Toolkit to create a JavaScript object helloWorldService for the service at /HelloWorldService using Dojo's RPC classes and providing them with SCA/SCADomain/HelloWorldService?smd for Simple Method Description (SMD) input.

It then creates a Dojo button that will envoke the getGreetings method of the helloWorldService object.
